# Outil de Facturation

Une application web de facturation développée avec React.

## Fonctionnalités
- Gestion des clients
- Création et suivi des factures
- Génération de PDF
- Tableau de bord avec statistiques

## Technologies
- React
- React Router
- TailwindCSS
- jsPDF
- recharts

## Installation
```bash
npm install
npm start
```
